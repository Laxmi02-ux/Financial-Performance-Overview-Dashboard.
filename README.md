# Financial-Performance-Overview-Dashboard.
Project Title: Financial Performance Overview Dashboard

Created a comprehensive financial performance analysis using Python, Pandas, and SQLite, followed by interactive visualization in Power BI. Cleaned and transformed sales and revenue datasets to generate insights into monthly trends, regional contributions, and profitability patterns. Built dynamic KPIs and charts to monitor revenue growth, top-performing products, and market segments. Enabled business stakeholders to track financial health and strategic opportunities efficiently through a visually engaging dashboard.

Key Skills:
Python, Pandas, SQLite, Power BI, Data Cleaning, KPI Analysis, Financial Analytics, Business Intelligence
